## 1\. 5-Minute Setup

### Step 1: Install (2 minutes)

**Windows:**

```cmd
pip install reportlab matplotlib Pillow requests numpy scikit-learn pandas scipy pyserial
python main_app.py
```

**macOS/Linux:**

```bash
pip3 install reportlab matplotlib Pillow requests numpy scikit-learn pandas scipy pyserial
python3 main_app.py
```

### Step 2: Add Your First Pool (1 minute)

1.  Click **Pool Management** tab
2.  Click **Add New Pool**
3.  Enter:
    -   Pool ID: `POOL001`
    -   Pool Name: `Main Pool`
    -   Volume: `20000` gallons
4.  Click **Save**

### Step 3: Enter First Reading (1 minute)

1.  Click **Chemistry Readings** tab
2.  Select your pool
3.  Enter test results:
    -   pH: `7.4`
    -   Free Chlorine: `2.0` ppm
    -   Total Alkalinity: `100` ppm
4.  Click **Save Reading**

### Step 4: Generate Your First Report (1 minute)

1.  Click **PDF Reports** tab
2.  Select your pool
3.  Choose "Last 7 days"
4.  Select "Executive Summary"
5.  Click **Generate Report**

**Done!** Your professional PDF report opens automatically! ðŸŽ‰

### 4.2 Alert Actions

**ðŸ”´ Critical Alert:**

-   Stop pool use immediately
-   Take corrective action
-   Retest after 30 minutes
-   Document in notes

**ðŸŸ¡ Warning Alert:**

-   Plan corrective action
-   Monitor closely
-   Retest within 24 hours
-   Add to maintenance schedule

**ðŸ”µ Info Alert:**

-   Note for reference
-   No immediate action
-   Review when convenient

### 4.3 Common Adjustments

**Lower pH:**

-   Add sodium bisulfate (dry acid)
-   Retest after 4 hours

**Raise pH:**

-   Add sodium carbonate (soda ash)
-   Retest after 4 hours

**Raise Chlorine:**

-   Add liquid/granular chlorine
-   Retest after 30 minutes

**Lower Chlorine:**

-   Stop adding chlorine
-   Let sunlight dissipate
-   Retest daily

### 5.2 Common Issues

**Application Won't Start:**

```
Solution:
1. Check Python version: python --version
2. Install dependencies: pip install -r requirements.txt
3. Check error messages
```

**PDF Reports Fail:**

```
Solution:
1. Install ReportLab: pip install reportlab
2. Check logo files exist
3. Verify write permissions
```

**Email Not Sending:**

```
Solution:
1. Check SMTP settings in Settings tab
2. For Gmail, use App Password
3. Test with "Send Test Email"
```

**Inventory/Shopping PDF Issues:**

```
Solution:
1. Ensure data files exist
2. Check branding configuration loaded
3. Verify file permissions
4. Test with sample data
```
